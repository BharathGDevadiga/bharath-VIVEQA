## Description

- **Waveform** – Displays the simulation results and verifies the functionality of the flip-flop.
- **Schematic** – Shows the synthesized RTL implementation of the flip-flop.
- **Device** – Displays the placement of the synthesized design on the target FPGA device.
- **Package** – Shows the FPGA package view with the input and output pin assignments.

These results verify the correct implementation and operation of the D, T, JK and SR Flip-Flops using Verilog HDL.
