# Day 04 - Verilog Codes

This folder contains the Verilog HDL programs for implementing the basic Flip-Flops.

## Files

- d_flipflop.v
- tb_d_flipflop.v

- t_flipflop.v
- tb_t_flipflop.v

- jk_flipflop.v
- tb_jk_flipflop.v
  
- sr_flipflop.v
- tb_sr_flipflop.v

## Description

- **d_flipflop.v** contains the Verilog module for the D Flip-Flop.
- **tb_d_flipflop.v** contains the testbench used to simulate and verify the D Flip-Flop.

- **t_flipflop.v** contains the Verilog module for the T Flip-Flop.
- **tb_t_flipflop.v** contains the testbench used to simulate and verify the T Flip-Flop.

- **jk_flipflop.v** contains the Verilog module for the JK Flip-Flop.
- **tb_jk_flipflop.v** contains the testbench used to simulate and verify the JK Flip-Flop.

- **sr_flipflop.v** contains the Verilog module for the SR Flip-Flop.
- **tb_sr_flipflop.v** contains the testbench used to simulate and verify the SR Flip-Flop.
