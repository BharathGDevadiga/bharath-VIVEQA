# Assignment – Sequential Circuits

This folder contains the assignment question paper and handwritten solutions for the **Sequential Circuits** topic in Digital Electronics.

## Files
- D4-Assignment_Questions.pdf
- D4-Assignment_Solutions.pdf

## Description

This assignment covers the analysis and design of sequential logic circuits using flip-flops and multiplexers. The topics include:

- Explaining the behavior of a given sequential logic circuit.
- Designing a **positive-edge triggered D Flip-Flop** using only **2:1 Multiplexers (MUXes)**.
- Designing a **negative-edge triggered T Flip-Flop** using only **2:1 Multiplexers (MUXes)**.
- Converting flip-flops by adding external logic gates:
  - Converting a **D Flip-Flop** into a **JK Flip-Flop**.
  - Converting a **T Flip-Flop** into a **D Flip-Flop**.
- Analyzing the output sequence of a **positive-edge triggered JK Flip-Flop** for a given input condition over multiple clock cycles.

**D4-Assignment_Questions.pdf** contains the original assignment questions, while **D4-Assignment_Solutions.pdf** contains handwritten, step-by-step solutions for all the problems.
