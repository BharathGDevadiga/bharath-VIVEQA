# Day 05 - Verilog Codes

This folder contains the Verilog HDL programs for implementing a **7-Segment Display Decoder**.

## Files

- segment_7disp.v
- tb_sevenseg.v

## Description

- **segment_7disp.v** contains the Verilog module for a 7-Segment Display Decoder. It converts a 4-bit binary input into the corresponding 7-segment display output for decimal digits.
- **tb_sevenseg.v** contains the testbench used to simulate and verify the functionality of the 7-Segment Display Decoder.
